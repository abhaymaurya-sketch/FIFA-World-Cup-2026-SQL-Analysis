# FIFA-World-Cup-2026-SQL-Analysis
SQL analysis of FIFA World Cup 2026 player performance using PostgreSQL.


## 📌 Project Overview

This project analyzes FIFA World Cup 2026 player performance using PostgreSQL. The dataset contains player-level match statistics, allowing advanced SQL analysis to identify top performers, evaluate team performance, and design custom KPIs.

---

## 🛠 Tools Used

- PostgreSQL
- pgAdmin 4
- SQL

---

## 📊 Dataset

- 54,600 Rows
- 75 Columns
- Player-wise match statistics
- Team performance metrics
- Tournament ratings

---

## 🧠 SQL Concepts Used

- Aggregate Functions
- GROUP BY
- HAVING
- ORDER BY
- Window Functions
- RANK()
- DENSE_RANK()
- PARTITION BY
- Common Table Expressions (CTEs)
- CASE Statements

---

## 📋 Business Questions Solved

- Top Goal Scorers
- Best Passers
- Highest Rated Players
- Goal Conversion Rate
- Team Rankings
- Best XI
- Player Performance Index (PPI)
- 30 SQL Business Questions

---

## ⭐ Custom KPI

Designed a custom **Player Performance Index (PPI)** using weighted performance metrics:

- 30% Player Rating
- 25% Performance Score
- 20% Goals
- 10% Assists
- 10% Pass Accuracy
- 5% Defensive Contribution
